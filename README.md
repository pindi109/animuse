# Animuse

> AI-powered storyboard video builder

Turn any topic into a long-form storyboarded video plan.

## Stack
| Layer | Technology |
|-------|-----------|
| Frontend | Flutter Web (Dart) |
| State Management | Riverpod |
| Navigation | go_router |
| HTTP | dio |
| Backend | Node.js REST API (separate repo) |

## Getting Started
```bash
git clone https://github.com/pindi109/animuse
cd animuse
flutter pub get
flutter run -d chrome
```

## Environment
```bash
flutter run -d chrome --dart-define=API_BASE_URL=http://localhost:3000
```
